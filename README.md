  https://doi.org/10.1109/ICTAI.2014.104

  https://doi.org/10.1142/S0218213016600046


## Compile & launch

The projects requires Java 1.6.
Confirmed to work with OpenJDK 1.8.

Compile:
```
javac -cp . ./v02/OknoGlowne.java
```

Launch
```
java -cp . v02.OknoGlowne
```
